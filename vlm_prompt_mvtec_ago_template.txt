Important principles:
- Focus on visual facts visible in the image. Do not invent hidden causes, materials, or defect mechanisms.
- Keep the background_description free of defect words, so it can be used to describe non-defective regions.
- Keep the defect_description focused only on the anomalous region: shape, size, color/contrast, edge quality, texture, depth, severity, and spatial extent.
- Describe the defect location relative to the object, not only relative to the image frame. Use anchors such as center, upper-left edge, lower-right surface, along the rim, across the printed area, near the cable end, on the nut face, etc.
- Prefer concise, concrete nouns and adjectives.
- If known metadata is provided (product_category, defect_type), use it when consistent with the image.
- If the defect is ambiguous or barely visible, set defect_presence to "uncertain" and note the limitation.
- If no defect is visible, set defect_type to "good", defect_presence to "none", and describe accordingly.
